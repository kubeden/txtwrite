# <img src="public/logo/logo.png" width="22" height="22"> TXTWrite

TXTWrite is an open-source markdown editor.

You can include it in your web project for free.

## Prerequisites

- node: v23.11.0
- npm: 11.6.2

## How to run

Go inside the root of the projects and execute:

```
npm install
npm run dev
```

The application runs on localhost:3000
